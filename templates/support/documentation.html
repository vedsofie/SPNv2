ELIXYS Software User Manual

The Software User Manual describes how to use the ELIXYS software, from programming sequences to explaining the run screen and log files.

ELIXYS FLEX/CHEM User Manual

The FLEX/CHEM User Manual describes how to use the instrument for automated synthesis. Pre-synthesis hardware preparation through exporting logs of past runs from the GUI are covered, including cassette setup.

ELIXYS PURE/FORM User Manual

The PURE/FORM User Manual introduces the new module and explains the workflow it was designed for. All features of the PURE/FORM instrument are detailed here and an breakdown of all PURE/FORM related software features are explained.

This document describes the facility requirements for a full ELIXYS platform: the FLEX/CHEM synthesizer, PURE/FORM module, and Control Box. It also describes what other items are required during the installation process.
